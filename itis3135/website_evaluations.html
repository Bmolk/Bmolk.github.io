Website Evaluations

Well-Done: Mythictrap

  • Contrast: This website uses a dark background with white text and has accents of yellow making information easy to read as well as making the site visually appealing.
  • Repetition: Elements on the website are consistent and the same layout is used for each page.
  • Alignment: The website has a centralized look with wider elemets being navigation menus.
  • Proximity: Elements on this website are in good proximity while keeping separate part distinguishable.

    Terrible: Arngren

  • Contrast: The entire website is white with a bunch of different text colors making is difficult to identify items.
  • Repetition: Layouts are not consistent from text color to element sizes which makes the page very confusing.
  • Alignment: There isn't any sense of alignment as elements are all over the page.
  • Proximity: Everything is far too close together causing the page to look chaotic.
